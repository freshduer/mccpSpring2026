# Poster submission folder

This folder holds your **Presentation 2 (poster presentation)** submission. The current poster content is based on the paper in **pre2/longer**: **LONGER: Scaling Up Long Sequence Modeling in Industrial Recommenders** (RecSys 2025).

## Files to include

| File | Description |
|------|-------------|
| `poster.html` | Poster (title, abstract, methodology, results, and figure placeholders filled from the LONGER paper) |
| `outlineScript.md` | Outline and script for the 2–3 minute presentation (LONGER-based) |
| `Q&A.md` | Q&A preparation with common questions and answers (LONGER-based) |
| `audio_link.md` | **Audio link** for your 2–3 minute recording (provide this link or the file link in the forum) |
| `images/` | Poster figures: convert PDFs from **pre2/longer/image/** to PNG and place them here; see `images/README.md` |

## Links to provide in your forum reply

1. **Poster link** — e.g. GitHub Pages URL for the poster, or the GitHub link to the poster HTML.
2. **Audio link** — The audio URL in `audio_link.md`, or the raw link to `audio_link.md`.
3. **Q&A file link** — GitHub link to `Q&A.md` (raw or repo view).
4. **(Optional) This folder path** — e.g.  
   `https://github.com/[your-username]/mccpSpring2026/tree/main/pre2/posterSubmission`
