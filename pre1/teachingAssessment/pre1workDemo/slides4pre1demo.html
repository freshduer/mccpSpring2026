The Rise of Multi-Agent AI
  • AI agents now collaborate in teams ‚Äî MetaGPT, AutoGen, ChatDev organize specialized agents to solve complex tasks
  • Each agent = Large Language Model + fixed instructions (role) + dynamic task input
  • Challenge: GPU memory is limited ‚Äî the system must decide which agent's "notes" to keep and which to discard
Key concept: "KV caches" are pre-processed notes that save the AI from re-reading the same instructions every time. Caching them is fast; recomputing them is slow.

What I Learned About Research Writing

LESSON 2
